# /init-docs - Inicializar Documentação Inteligente

Este comando inicia o processo de estruturação de documentação para um novo projeto.

### 🧠 Fase 1: Descoberta (Brainstorming)
**OBRIGATÓRIO:** Antes de criar qualquer arquivo, a IA deve utilizar a skill `brainstorming` para realizar o **Socratic Discovery**.
- Faça ao menos 3 perguntas estratégicas para entender:
    1. O objetivo principal do projeto.
    2. O público-alvo ou usuários finais.
    3. As tecnologias principais que serão utilizadas (se ainda não definidas).

### 📁 Fase 2: Geração de Templates
Após as respostas do usuário, gere os seguintes arquivos preenchendo as informações coletadas:

1. **README.md**: Visão geral personalizada.
2. **ARQUITETURA.md**: Mapa de pastas baseado na estrutura detectada.
3. **COMANDOS.md**: Scripts do `package.json`, comandos Git e Gestão de Skills.
4. **CHECKLIST_NOVO.md**: Guia de onboarding para o projeto.

---

## Como usar

Peça ao agente:
- `/init-docs`
- "quero iniciar um projeto novo"
- "gere a documentação inicial pesquisando sobre o projeto"
