# 🛠️ Dashboard Elite - Guia de Comandos

Este guia centraliza as tecnologias e comandos necessários para a manutenção do **PortalGov Admin Dashboard**.

## 🌌 Stack Tecnológica de Alta Performance (2025)
Abaixo as tecnologias que sustentam o ecossistema administrativo:

| Camada | Tecnologia | Propósito |
| :--- | :--- | :--- |
| **Framework** | **Next.js 15.1** (App Router) | Motor SSR/ISR de última geração |
| **Estilização** | **Tailwind CSS 4.0** | Design System CSS-First e alta performance |
| **Backend/DB** | **Supabase** (PostgreSQL) | Banco de dados, Auth e Storage em tempo real |
| **Cache/Sync** | **TanStack Query v5** | Sincronização de estado servidor/cliente |
| **Estado Local** | **Zustand** | Gestão de UI e preferências globais |
| **Animações** | **Framer Motion** | Micro-interações e transições fluídas |
| **Ícones** | **Lucide React** | Biblioteca de ícones institucionais |
| **Comunicação** | **Axios** | Cliente HTTP robusto para integrações |

## 🚀 1. Servidor & Desenvolvimento
Você pode rodar estes comandos em **QUALQUER terminal** aberto na pasta do projeto.

| Objetivo | Comando Rápido | Descrição |
| :--- | :--- | :--- |
| **Iniciar Portal** | `npm run dev` | Liga o site na porta 3000. |
| **Reset / Erro** | `npm run reset` | Limpa processos e religa o site (Use se der erro). |
| **Build Final** | `npm run build` | Prepara a versão de produção. |

> [!TIP]
> Agora você não precisa mais entrar na pasta `portalgov-v3`. O comando funciona da pasta raiz!

## 🪄 2. Automações de IA (Comandos `/`)
Digite `/` no chat para disparar processos automáticos.

* **`/enhance`**: Melhora páginas existentes ou adiciona novas funções.
* **`/ui-ux-pro-max`**: Modernização visual profunda para design de elite.
* **`/debug`**: Investigação avançada para encontrar a causa raiz de erros.
* **`/status`**: Mostra o progresso atual e o que as IAs estão fazendo.
* **`/orchestrate`**: Missões complexas que precisam de vários especialistas.
